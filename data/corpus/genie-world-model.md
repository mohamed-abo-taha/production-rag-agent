# Genie (world model)

Source: Wikipedia (https://en.wikipedia.org/wiki/Genie_%28world_model%29)

Genie, Genie 2 and Genie 3 are world models developed by Google DeepMind that generates interactive virtual worlds. It was initially trained on video game footage.  The first version of Genie was capable of generating two-dimensional interactive environments. Its successor, Genie 2, expanded these capabilities to generate three-dimensional environments. Genie 3 has higher-resolution world generations and increased memory capability compared to its predecessors. Waymo has adopted Genie 3 and created a specialized world model, the Waymo World Model, for autonomous driving simulation.  Project Genie is a website that allows subscribers to use Genie 3 to generate worlds.  Genie is used in video game design, training AI agents, and autonomous vehicles.


== Versions ==


=== Original ===
The original Genie model, introduced in March 2024, was only capable of generating two-dimensional environments. It generates frames of the world from interactive input and previous frames, and was trained on video game footage. The original Genie generated worlds at only one frame per second, which made worlds unplayable as video games.


=== Genie 2 ===
Genie 2, introduced in December 2024, had the capability of generating three-dimensional environments as well as two-dimensional ones. It only could generate worlds in the isometric and first-person perspectives, and generated worlds last 10–20 seconds. Compared to Oasis, Genie 2 has greater memory and rendering capabilities. Genie 2 could generate worlds in 360p resolution.


=== Genie 3 ===
Genie 3, introduced in August 2025, is capable of generating and modifying interactive worlds in real-time. It has a memory of one minute, which is higher than Genie 2's 10–20 seconds. Genie 3 generates worlds in 720p at 24 frames per second. Genie 3 has an integration with Google Street View, which allows it to generate streets.


== Applications ==
Genie is intended for applications in robotics and simulations. Genie has been used in video game design, including creating imitations of Nintendo's games Super Mario 64 and The Legend of Zelda: Breath of the Wild.  It has also been used in training AI agents.


=== Self-driving cars ===
Waymo adopted Genie 3 for simulating edge cases for robotaxis, creating a variant called the Waymo World Model. It outputs lidar at four times the speed of Genie 3, and creates simulations that are more realistic than Genie 3's. It takes video input recorded with a dashcam or phone.
The Waymo World Model has three types of simulation controls:

Driving action control controls how it responds to inputs.
Scene layout control controls the road's layout and the actions of the other cars.
Language control controls environmental conditions.
The Waymo World Model helped introduce Waymo's robotaxis to 11 cities in the United States.


== Project Genie ==

Project Genie is a website that allows Google AI Ultra subscribers and are over the age of 18 to access Genie 3. Project Genie was released on January 29, 2026. Project Genie is accessible via Google Labs.
It has a 60-second limit on world exploration. In generated worlds, the WASD keys are used to move, the arrow keys are used to turn the camera, and the space bar is used to ascend. Project Genie is integrated with Google Street View, allowing it to generate streets accurately.
After the release of Project Genie, shares of video game producers decreased significantly, with Take-Two Interactive decreasing by 9.3%, Roblox Corporation decreasing by 15%, Unity Software decreasing by 21%, and CD Projekt decreasing by 8%.
